## [GREEN · GO] arxiv-openproblem:1712.01960v1 (composite 4.9357, openness conf high)
**Problem:** Determine the worst-case minimal distortion required to embed an n-point diversity into l1 (the diversity analogue of the classical metric embedding into l1). In particular, give bounds or exact values for the function f(n) = max_{diversity on n points} (minimal distortion to embed that diversity into l1).

**Win condition:** Produce one or more explicit, machine-checkable certificates for small n (e.g. n ≤ 8–12): either (A) a diversity instance together with a proof/certificate (embedding map into l1 and computed distortion) showing a new higher lower bound on worst-case distortion for that n, or (B) an explicit embedding algorithm and concrete embeddings achieving a strictly better upper bound for some n. Certificates should be runnable checks (LP/ILP instances, explicit coordinate vectors, and computed distortion values), and ideally a formalized Lean statement verifying the distortion computation for at least one nontrivial n.

**Still open / closest prior:** open=True — Closest prior is Bryant–Tupper’s 2017 open-problem note: general n-point diversities have only the trivial O(n) upper bound and an inherited Ω(log n) lower bound from metric L1 embedding; they explicitly say O(log n) is not ruled out when dimension is unrestricted. ([arxiv.org](https://arxiv.org/abs/1712.01960)) The strongest later adjacent result I found is Jozefiak–Shepherd 2023: O(log n) for diameter diversities, improved hypergraph-sparsest-cut applications, but still states the smallest known distortion for arbitrary diversities is O(n). ([arxiv.org](https://arxiv.org/abs/2303.04199)) Other close papers cover special cones/classes—symmetric diversities have constant distortion, negative-type diversities have their own L1 lower-bound theory, and 2024 linear/sublinear diversity embeddability is characterized via negative type/Minkowski circumradius—but none appears to give exact small-n f(n) tables or certified extremal instances. ([arxiv.org](https://arxiv.org/abs/1604.01863)) ([arxiv.org](https://arxiv.org/abs/1809.06523)) ([arxiv.org](https://arxiv.org/abs/2412.07092)) A May 2026 ResearchGate note titled “Diversity Fréchet Embedding” is explicitly AI-generated and “not very helpful,” so I would not count it as closing the gap. ([researchgate.net](https://www.researchgate.net/publication/405390936_Diversity_Frechet_Embedding?utm_source=openai))

**Why not already done:** It survived because the global problem is a quantifier-over-a-cone problem, not because the fixed-instance computation is hard. For a fixed rational diversity δ, minimal L1 distortion is a clean cut-cone LP using nonnegative split weights, and Bryant–Tupper already note finite L1 diversities are nonnegative combinations of split diversities. ([d-nb.info](https://d-nb.info/1325881090/34?utm_source=openai)) But maximizing that LP optimum over all valid diversities is a high-dimensional polyhedral/non-smooth search problem with many diversity inequalities and many cuts; prior work pursued asymptotic embeddings and special classes, not exact computational extremals. This is precisely the niche where a small AI+LP/SAT team can add value: find a rational δ, certify it is a diversity, solve the distortion LP exactly, and ship primal/dual certificates.

**1-week recommendation:** GO (engine B)

**First concrete step:** Day 1: implement the exact rational fixed-instance certificate pipeline for n=6 first. Enumerate all subsets S with |S|≥2 and all cuts U up to complement; verify diversity by δ(A∪B)≤δ(A)+δ(B) for intersecting A,B; solve the LP maximize t subject to t·δ(S) ≤ Σ_U λ_U·1[U cuts S] ≤ δ(S), λ_U≥0. Export (i) δ-values, (ii) primal split weights λ giving the best embedding, (iii) dual/Farkas certificate proving no larger t, all as rationals. Then wrap OpenEvolve/random rational polytope walks around δ to maximize 1/t, with filters excluding symmetric, diameter, Steiner, and obvious negative-type-only examples.

**Key risk:** The main risk is not openness; it is significance. A certificate for n=3 or n=4, or a symmetric/diameter-looking example, will be dismissed as derivable from existing special-class theory. The week succeeds only if the search finds a genuinely non-symmetric n≥6 instance with a rational LP optimum beating the obvious sanity-check examples and with a dual certificate short enough to audit.
