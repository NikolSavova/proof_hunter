**Problem:** For integers e,m with 3 ≤ e < m, determine R(e,m) := sup{ ρ(Γ) | edim(Γ) = e+1, mult(Γ) = m }, the supremum (maximum if attained) number of relations in a minimal presentation among numerical semigroups with embedding dimension e+1 and multiplicity m.

([arxiv.org](https://arxiv.org/pdf/2405.19810)) The arXiv open-problems article also records the same landscape: R(e,e+4)=R(e,e+5)=binom(e+1,2)+1, R(e,e+6)=binom(e+1,2)+2, R(e,e+7)=binom(e+1,2)+4 for e≥9 but not for e=3,4,5, and the central edim-4 conjecture R(3,m)=⌊2+2√m⌋. ([re.public.polimi.it](https://re.public.polimi.it/retrieve/dfe7de85-1ab0-40ea-8036-d0c1a392e077/Open%20problems%20on%20relations%20of%20numerical%20semigroups.pdf)) Elmacioglu--Hilmer--O’Neill--Okandan--Park-Kaufmann is the key computational/combinatorial prior for the edim-4 case: it conjectures that its edim-4 families attain all possible minimal-presentation cardinalities, says this was computationally verified for m≤42, and gives a construction with edim=4, multiplicity m, and η for every η≥6 satisfying 4m≥(η−2)^2, i.e. the lower-bound side of η≤⌊2+2√m⌋. ## 2. [AMBER] arxiv-openproblem:2511.18217v1#2  (composite 4.7643, confidence medium, still_open=True)

**Problem:** Let M be the boundary of the R-neighborhood of a segment (an R-stadium). For a given r>0 describe all maximal distance minimizers (r-minimizers) for M. Known: for r < R < 5r every MDM is a horseshoe, while for R < 1.75 r the solution is not a horseshoe. Open: (a) Find the supremum C such that there exists an r-minimizer for a C r-neighborhood which is not a horseshoe. (b) Find the infimum c such that for any closed curve with minimal radius of curvature c r every MDM is a horseshoe.

**Win condition:** Either (i) produce an explicit counterexample stadium (explicit R/r numeric value and geometry) together with a certificate (piecewise-linear or analytic description of an r-minimizer that is not a horseshoe) plus a short rigorous verification checkable by computation/Lean; or (ii) prove a rigorous inequality establishing C (or c) with numeric bounds (e.g. show C ≤ C0 or c ≥ c0) and provide a Lean-checkable proof/verification routine for the bound.

**Closest prior / novelty check:** Closest prior is Cherkashin--Teplitskaya, “On the horseshoe conjecture for maximal distance minimizers”: for every closed convex curve with minimal radius of curvature R and r < R/5, all minimizers are horseshoes; for the circle they improve this to r < R/4.98. They also give the basic long-stadium counterexample showing horseshoes can fail when R/r is close to 1. ([numdam.org](https://www.numdam.org/item/10.1051/cocv/2017025.pdf)) Cherkashin’s dissertation/overview later states the sharper known stadium construction: for a sufficiently long stadium, if R < 1.75 r, there is a connected covering set shorter than any horseshoe, hence a global MDM cannot be a horseshoe; it lists “describe r-minimizers for a given stadium” and improving the 5r constant as open. ([ras.nacid.bg](https://ras.nacid.bg/api/reg/FilesStorage?dbId=2&fileName=Thesis-Dissertation+ENG-D.Cherkashin.pdf&key=e0c5b9c3-810e-446a-96f0-463c2bae3b44&mimeType=application%2Fpdf)) The 2025 arXiv open-problems paper repeats exactly this status: stadium open for r < R < 5r, known non-horseshoe for R < 1.75r, known horseshoe for R > 5r, with the two threshold questions still open. ([researchgate.net](https://www.researchgate.net/publication/397934307_Open_problems_on_Steiner_trees_and_maximal_distance_minimizers?utm_source=openai)) Current bracket: for stadiums, 1.75 ≤ C ≤ 5; for arbitrary closed curves, 1.75 ≤ c ≤ 5, modulo strict endpoint conventions.

**Novelty notes / residual risk:** Not killed as “already solved”: I found no later paper closing the stadium threshold or the general curvature threshold. The danger is instead that the easiest publishable-looking win is already prior art: an explicit non-horseshoe stadium for any numeric R/r < 1.75 is not new, because the thesis/open-problem text already asserts that construction. A new lower-bound result must beat 1.75, or give a materially more explicit/computer-checkable certificate for a value not previously certified; a new upper-bound result must improve 5 or give a rigorous restricted-class theorem. The statement’s win condition is a bit ambiguous: if it accepts “some explicit counterexample,” then it is already satisfied by prior art; if it requires a new threshold improvement, it remains open but nontrivial.

**LLM-saturation:** I found no evidence that this exact MDM R-stadium problem appears in GPT-Erdos, DeepMind Formal Conjectures, GitHub formalization repos, or the AlphaEvolve public problem list. Searches for exact phrases plus GitHub/Lean/formal terms produced no relevant hits. AlphaEvolve’s 2025 paper and public repository cover 67 math problems across analysis, combinatorics, geometry, and number theory, but search hits from that repository were unrelated to maximal-distance minimizers/stadiums. ([arxiv.org](https://arxiv.org/abs/2511.02864?utm_source=openai)) So LLM saturation appears low; the adjacent risk is human-literature saturation by the Cherkashin--Teplitskaya school, not frontier-lab duplication.

**Attack sketch:** Best one-week attack is a lower-bound construction, not an exact solution. Normalize r = 1 and write λ = R/r. For a long stadium of length T, any horseshoe has asymptotic length 2T + O_λ(1), as used explicitly in the prior stadium counterexamples. ([numdam.org](https://www.numdam.org/item/10.1051/cocv/2017025.pdf)) Reduce to an infinite two-line strip y = ±λ: search for a periodic connected network cell of horizontal period p whose 1-neighborhood covers both boundary lines in the cell and whose length is < 2p − δ. Repeating the cell and adding O_λ(1) end-cap gadgets gives a long-stadium competitor shorter than every horseshoe; existence of an MDM then implies at least one non-horseshoe MDM. Certificate format: finite graph with rational/algebraic vertex coordinates, straight edges, periodic glue data, and interval-arithmetic proof of (i) coverage of the two lines and end caps by distance-to-segment inequalities, (ii) connectedness, and (iii) total length < horseshoe lower bound. Engine A: rederive the strip-repetition lemma and encode it in Lean/interval arithmetic. Engine B: evolutionary continuous search over periodic Steiner-like graph topologies with degree ≤3 and 120-degree bias; then snap to algebraic/rational coordinates and verify by interval arithmetic/CAD. Also run SAT/MILP-style topology enumeration for small cells to avoid rediscovering only the known 1.75 construction. Realistic in ~1 week: rediscover the 1.75 construction and maybe find a modest C > 1.75 lower-bound certificate. Not realistic in ~1 week: proving the true supremum C, improving the universal upper bound 5 in full generality, or fully classifying all stadium MDMs.
