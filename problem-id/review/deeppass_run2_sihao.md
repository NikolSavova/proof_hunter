**Still open / closest prior:** open=True — The nearest prior is the EHOOPK/Algebraic Combinatorics paper, which for embedding dimension 4 proposes the candidate maximum R(3,m)=⌊2+2√m⌋, gives constructions attaining that value, and reports computational verification of the relevant e=4 families through m≤42. ([numdam.org](https://numdam.org/item/10.5802/alco.354.pdf)) A second close prior is Brower–McDonough–O’Neill 2025: it introduces Kunz-fan walking, reports that all k=3, m≤42 Kunz-fan faces were computed, and proves the important reduction that a maximum is achieved by a staircase/unique-Apéry-expression Kunz nilsemigroup with the same atom set. ([alco.centre-mersenne.org](https://alco.centre-mersenne.org/item/10.5802/alco.425.pdf)) ([alco.centre-mersenne.org](https://alco.centre-mersenne.org/item/10.5802/alco.425.pdf)) For general e,m, Moscariello–Sammartano’s BLMS paper gives the main upper-bound framework C(e,m), proves equality for m−e≤6 and asymptotic/non-sharpness phenomena, and explicitly says the complete problem is still out of reach with current tools. ([arxiv.org](https://arxiv.org/pdf/2405.19810)) ([arxiv.org](https://arxiv.org/pdf/2406.00790v2))

**Why not already done:** The global problem is genuinely open, but the easy computational niche is narrower than the win condition suggests. For user e=3, the conjectural formula and lower-bound constructions are already known, and existing computation reached m≤42; pushing to m≤50 is too incremental unless accompanied by a very clean independent certificate, while m≤200 requires avoiding full Kunz-face enumeration because the 2025 Kunz-fan computation already ran out of memory at m=43 for k=3. ([alco.centre-mersenne.org](https://alco.centre-mersenne.org/item/10.5802/alco.425.pdf)) For e=4,5 the obstacle is not computing ρ for one semigroup—GAP/NumericalSgps can do that—but proving maximality: MS25 ties the upper bounds to Hilbert functions, compressed lex ideals, additive-combinatorics constraints, and EGH/LPP-type phenomena, so observed maxima without upper certificates will be hard to sell as a publishable result. ([arxiv.org](https://arxiv.org/pdf/2405.19810)) ([arxiv.org](https://arxiv.org/pdf/2405.19810))

**First concrete step:** Day 1: do not enumerate numerical semigroups. Encode the post-2025 reduced problem: staircase Kunz nilsemigroup/order-ideal certificates. For fixed user e and m, variables are an order ideal O⊂N^e of size m with injective residue map O→Z/mZ, atom residues generating Z/mZ, and boundary/minimal-excluded monomials counted as outer Betti elements; add SAT/CP-SAT clauses to prove no boundary size exceeds the candidate. First target user e=3, m=43–80, because this directly extends the published m≤42 verification and has a clear certificate format: candidate semigroup generators plus UNSAT proof for boundary ≥⌊2+2√m⌋+1.

**Still open / closest prior:** open=True — Closest prior is Cherkashin–Teplitskaya’s horseshoe theorem: every minimizer is a horseshoe for a closed convex curve with minimal curvature radius at least 5r, and for a circle the constant is improved to 4.98r. The stadium-specific lower counterexample is the Σ0 periodic/local-Steiner tile: for a long R-stadium with R<1.75r it gives a covering connected set shorter than any horseshoe, hence no horseshoe can be globally minimal. The November 2025 open-problem note explicitly leaves the R-stadium case open for r<R<5r and asks for the supremum C / curvature threshold c; it also notes that stadium minimizers consist of line segments and arcs parallel to the stadium arcs. ([numdam.org](https://www.numdam.org/item/10.1051/cocv/2017025.pdf))

The lower-bound side is more plausible but still nontrivial: one must beat the 1.75 periodic tile by solving a periodic strip-covering network problem and then certify coverage and length with interval/rational inequalities. It survived because finding a better-looking drawing is not enough; the competitor must cover the whole long stadium and have asymptotic length per unit strictly below 2, while the proof must be robust under end effects.

**Key risk:** No periodic competitor actually beats 1.75, or the search finds only a numerical candidate whose coverage/length margin disappears under rigorous interval certification. The upper-bound route should be treated as no-go for one week.
